# Cintia Larizzati - Spec-Driven Development (SDD)

## Meta
Mejorar la landing page de ventas en Hotmart y estructurar el curso utilizando los estándares de diseño premium de Multiversa Group (UI/UX Pro Max).

## Fase 1: Análisis de Ingestión
- **Plataforma:** Hotmart.
- **Público Objetivo:** Mujeres/Profesionales (requiere validación con perfil de Cintia).
- **Necesidades Detectadas:** Mejorar tasas de conversión, diseño "Glassmorphism" y modo oscuro elegante, llamadas a la acción (CTAs) de alta visibilidad.

## Fase 2: Orquestación de Skills
Para este proyecto, el enjambre de Multiversa utilizará los siguientes skills:
1. `ui-ux-pro-max`: Para definir la paleta de colores, tipografía (interacciones premium, micro-animaciones).
2. `frontend-design`: Para la construcción de componentes web nativos (HTML/CSS/JS) sin dependencias externas pesadas.
3. `MiroFish` (Simulación): Antes de lanzar, simularemos el impacto de 3 tipos de títulos diferentes en el público objetivo para seleccionar el mejor.

## Fase 3: Tareas de Ejecución
- [ ] Construir layout base en `index.html`.
- [ ] Implementar sistema de tokens de diseño en `styles.css`.
- [ ] Preparar integración de checkout de Hotmart en CTAs.
