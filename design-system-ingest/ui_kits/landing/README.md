# Multiversa Landing — UI Kit

Hi-fi recreation of the `multiversa-next` marketing page. Reproduces Hero, Triad, Pricing ladder, Comparison table, and Footer.

**Source:** `@multiversa/core/apps/multiversa-next/src/components/landing/*.tsx`

## Files
- `index.html` — clickable landing preview
- `Hero.jsx` · `Triad.jsx` · `PricingLadder.jsx` · `Benchmark.jsx` · `Footer.jsx`

## Components demonstrated
- Pill status tag (chartreuse dot + mono label)
- Two-beat serif/italic headline
- Primary pill CTA + glass CTA duo
- Agent cards (violet/teal/rose glow)
- Tier cards with step label, JTBD quote, feature bullets
- Benchmark comparison table (Spark column highlighted)
- Footer with MULTIVERSA chrome wordmark

Not 1:1 feature-complete. No framer-motion / GSAP — CSS transitions only.
