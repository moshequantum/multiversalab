## 🌐 Overview / Visión General

🇺🇸 **Multiversa Lab** is the open-source technical foundation where all source code, AI orchestration logic, and experimental development environments run. It is strictly separated from **Multiversa Group** (the business and consulting entity). The Lab is our engine room for the Intelligent Personal OS.

🇲🇽 **Multiversa Lab** es la base técnica de código abierto donde se ejecuta todo el código fuente, la lógica de orquestación de IA y los entornos de desarrollo experimental. Está estrictamente separado de **Multiversa Group** (la entidad comercial y de consultoría). El Lab es nuestro cuarto de máquinas para el Sistema Operativo Personal Inteligente.

## 🏗 Core Architecture / Arquitectura Core

We follow the **Liquid Glass Pro + Onion UI** design system and the **Triad Protocol** for agent orchestration. <br>
*Seguimos el sistema de diseño **Liquid Glass Pro + Onion UI** y el **Triad Protocol** para la orquestación de agentes.*

- **◎ MultiversaOS (Shared Context):** Synchronizes state universally. / *Sincroniza el estado universalmente.*
- **⬡ Memory Persistence (Engram):** Logs architectural decisions. / *Registra decisiones arquitectónicas.*
- **✦ Knowledge Mapping (Graphify):** Semantic ingestion of active repositories. / *Ingesta semántica de repositorios activos.*
- **⚡ Operational Harness (Gentle SDD):** Enforces disciplined software development pipelines. / *Fuerza pipelines de desarrollo de software disciplinados.*
